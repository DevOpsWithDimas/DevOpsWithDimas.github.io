    Insecure Registry

    Docker for DevOps engineer April 24, 2021 1 minute read

    Hai semuanya, sebelumnya kita sudah memahami apa itu Docker Registry, sekarang kita akan membahas tentang Insecure Registry.

    Nah yang jadi pertanyaaan apa itu Insecure Registry?. Insecure Registry sama halnya dengan Docker Hub yaitu bertujuan menyimpan docker images tetapi disini insecure registry lebih flexible artinya kita bisa tentukan mulai dari lokasi registry (on-premise, cloud), private atau public docker repository dan lain-lain. Adapun materi yang akan kita bahas kali ini yaitu

    1. Macam-Macam Provider Insecure Registry
    2. Manfaat menggunakan Insecure Registry
      1. Proxy Repository for saving bandwidth
      2. Unlimited private repository
      3. Push & Pull Docker Images more faster than Docker Hub
