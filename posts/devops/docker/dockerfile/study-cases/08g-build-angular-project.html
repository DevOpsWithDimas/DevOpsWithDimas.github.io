    Setup SDK for Angular Project

    Untuk menggunakan angular project, kita memerlukan package dependency management yaitu

    1. nodeJs, kita tidak akan menggunakan nodejs secara langsung tetapi kita menbutuhkannya untuk melakukan compile source angular menjadi PWA (Progressive Web Apps) dan SPA (Single Page Applications).
    2. nvm adalah Node Version Manager
      • nvm for windows,
      • nvm for linux/mac
    3. npm, adalah Node Package manager

    Untuk menginstall nodeJs dan karena version dari NodeJs itu cepat sekali untuk update-nya serta cepat deprecated jadi kita di sarankan menggunakan nvm yaitu NodeJs Version Management dengan tujuannya supaya kita mudah switch between version nodeJs.
